# SCZip

SCZip は SoundCloud の楽曲ページからジャケット画像と音声を ZIP で保存する Chrome 拡張機能です。

## 使い方

1. SoundCloud で楽曲ページを開く
2. 拡張機能のポップアップを開く
3. 「ジャケット+音声をZIPで保存」ボタンを押す

## 機能

- サポートされた楽曲の音声とジャケットを ZIP にまとめて保存
- ダウンロード許可がない曲でも可能な限りストリーミング音声を取得
- ジャケット画像を JPG/PNG で選択可能
- 保存前にファイル名を確認できるオプション

## 注意

- SoundCloud の利用規約に従ってください
- 一部の楽曲はダウンロード不可のため、取得できない場合があります
